# Prompt 06 - Charts and i18n

## Objetivo

Mejorar visualizacion y lenguaje sin romper el rendimiento.

## Limites

- no redisenar toda la arquitectura
- no introducir dependencias pesadas sin justificar

## Entregables

- charts tactiles estables
- copy mas claro en ingles y espanol
- estados vacios y degradados pulidos

## Criterio de exito

- el usuario entiende que ve en cada chart
- no hay mezcla de idiomas
