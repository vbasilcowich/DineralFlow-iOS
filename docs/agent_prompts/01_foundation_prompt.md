# Prompt 01 - Foundation

## Objetivo

Dejar el repo iOS con estructura estable, scripts de calidad y una shell minima alineada con DineralFlow.

## Limites

- no implementar networking aun
- no crear charts reales
- tocar solo `README`, config, navegacion base y archivos de arranque

## Entregables

- identidad visual minima de producto
- scripts de `lint`, `typecheck`, `test`, `doctor`
- navegacion base sin pantallas demo de Expo

## Criterio de exito

- el repo compila
- los textos base ya no parecen una plantilla de Expo
- la estructura queda lista para conectar datos reales
