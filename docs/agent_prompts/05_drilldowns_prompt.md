# Prompt 05 - Drilldowns

## Objetivo

Portar las subpantallas de mayor valor informativo.

## Limites

- mantener navegacion y copy sencillos
- no replicar tablas web densas sin adaptacion

## Entregables

- provenance
- basket detail
- drivers
- frictions
- asset detail

## Criterio de exito

- cada pantalla explica mejor la lectura sin sobrecargar
- los datos siguen viniendo del backend real
